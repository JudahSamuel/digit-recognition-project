\# 🧠 Advanced Handwritten Digit Recognition System



\## 📌 Overview



This project uses a Convolutional Neural Network (CNN) to recognize handwritten digits (0–9) from user input.



\## 🚀 Features



\* Draw digit using canvas

\* Upload handwritten image

\* Real-time prediction

\* Confidence score

\* Probability visualization



\## 🧠 Technologies Used



\* Python

\* TensorFlow / Keras

\* Streamlit

\* NumPy

\* Matplotlib



\## 📊 Model



\* CNN trained on MNIST dataset

\* Accuracy: \~99%



\## ▶️ Run the Project



```bash

pip install -r requirements.txt

streamlit run app.py

```



\## ⚠️ Note



\* Only single digit (0–9) supported

\* Multiple digits may give incorrect results
